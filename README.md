## 📖 Giới thiệu
Dự án này nhằm xây dựng một hệ thống quản lý thư viện sách sử dụng Node.js, giúp quản lý sách, người mượn và các hoạt động liên quan một cách hiệu quả.

## 🚀 Cách chạy dự án
```bash
# Cài đặt dependencies
npm install

# Chạy server
npm start
```
